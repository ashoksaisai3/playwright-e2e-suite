# DemoQA Playwright Automation Framework

## 📌 Tech Stack
- Playwright
- TypeScript
- Node.js

---

## 📁 Project Structure

```plaintext
demoqa
 ├── src
 │    ├── fixtures
 │    ├── pages
 │    └── tests
 ├── testdata.csv
 ├── playwright.config.ts
 ├── tsconfig.json
```

---

## 🚀 Installation

```bash
npm install
```

Install Playwright browsers:

```bash
npx playwright install
```

---

## ▶️ Run Tests

Run all tests:

```bash
npx playwright test
```

Run specific test:

```bash
npx playwright test sample.spec.ts
```

---

## ✅ Implemented Scenarios

### 1. Practice Form Submission
- Fill form
- Submit form
- Validate success modal

### 2. Text Box Validation
- Fill text box form
- Validate output data

---

## 📊 Framework Features
- Page Object Model (POM)
- Data-driven testing using CSV
- Playwright Fixtures
- Cross-browser support
- HTML reporting

---

## ⚠️ Troubleshooting

If browsers are missing:

```bash
npx playwright install
```

If dependencies are missing:

```bash
npm install
```
